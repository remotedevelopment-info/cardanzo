Cardanzo.app

Vereion 0.0.0

What is it?

Nothing, yet. It will deploy existing DAPS using Cardano. It will trade with NFTs. It will employ some sort of AI engine, probably written in Mojo, open-source, github hosted.

Who is it?

It will be open-sorce so developers who are interested in working on this can join in the fun via github.

Why is it?

ADA is a currency that can already be used in a metaverse, for NFTs, by employing AI create a simple ecosystem for rewarding developers as owners: instead of being paid nothing for open-source projects or offering employment, projects are assigned a number of NFTs which cost (for example) 100 ADA each. Developers can obtain the NFTs and that establishes the capitial base for the project and its ownership. AI will value the project and guide the value of the NFT shares. Users can also buy Client NFTs to use the project. The idea is to create a remote development ecosystem, mediated by an AI engine to create useful things.

Cardanzo is a project fostered by Software Antelope Ltd - a company registered in the UK.
