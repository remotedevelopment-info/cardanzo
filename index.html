What is it?

Nothing, yet. It may deploy existing DAPS using Cardano. It may trade with NFTs. It could employ some sort of AI engine, probably written in Mojo, open-source, github hosted.

Who is it?

It will be open-sorce so developers who are interested in working on this can join in the fun via github. At the moment there are no developers working on this. If you want, then follow our github.

Why is it?

ADA is a currency that can already be used in a metaverse, for NFTs, by employing AI create a simple ecosystem for rewarding developers as owners: instead of being paid nothing for open-source projects or offering employment, projects are assigned a number of NFTs which cost (for example) 100 ADA each.

How?

Capitalisation: Developers obtain NFTs to establish the capital base for a project and its ownership. AI values projects to guide the value of the NFT shares. Users can also buy Client NFTs to use the project. The idea is to create a remote development ecosystem, mediated by an AI engine to create useful things.
